# Chatm

A simple chat app featuring webm integration!

### Dependencies

 - [MongoDB][1]

 - [Node.js][2]

Also note  `mongo` should be running as a service.

### Installation

Simply run 

 ```
	$ npm install
	$ npm start
 ```

then point your browser to `http://localhost:3000`

[1]: https://nodejs.org/en/
[2]: https://www.mongodb.com/
